# Arabic Keyboard Mapper

Version 1.0.0

A keyboard extension that converts English text to Arabic characters.

## What is this?

This extension maps QWERTY keyboard keys to Arabic characters. Select English text and press Ctrl+Shift+A to convert it to Arabic.

## Installation

1. Open Chrome and go to chrome://extensions/
2. Enable Developer Mode (top right)
3. Download this project folder
4. Drag and drop the folder onto the extensions page
5. Confirm the installation

## How to Use

1. Click the extension icon in your browser toolbar
2. Toggle the extension ON/OFF
3. Select English text on any website
4. Press Ctrl+Shift+A to convert selected text to Arabic
5. The extension only works when toggled ON

## Features

- Works on all websites (Gmail, YouTube, Twitter, Facebook, etc)
- Simple toggle on/off interface
- One-click conversion with keyboard shortcut
- Supports text inputs, textareas, and rich text editors
